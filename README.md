# ARPAN — Blockchain-Based Donation System

## 📌 Project Overview
ARPAN is a decentralized donation platform built using Blockchain technology.  
It allows users to create fundraising campaigns and donate securely using Ethereum (ETH).  
All transactions are executed through smart contracts, ensuring transparency, security, and immutability.

---

## 🚨 Problem Statement
Traditional donation platforms suffer from:
- Lack of transparency
- Centralized control
- Possibility of fund misuse
- Hidden transaction records

ARPAN solves these issues using blockchain-based smart contracts where every transaction is publicly verifiable.

---

## ✅ Proposed Solution
The system uses Ethereum blockchain to:
- Remove intermediaries
- Enable secure peer-to-peer donations
- Store transactions permanently on blockchain
- Provide transparent fund tracking
